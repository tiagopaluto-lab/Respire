Respire — Prototype d'application (statique)
-------------------------------------------
Contenu:
- index.html : page principale (FR)
- styles.css : styles
- app.js : logique (stockage local)
- manifest.json : manifest PWA basique
- README.txt : ce fichier

Mode d'emploi:
1) Dézippe respire_app.zip
2) Ouvre index.html dans ton navigateur (Chrome/Edge/Firefox)
3) Tu peux "installer" l'app (Chrome) pour l'ajouter à l'écran d'accueil.
4) Toutes les données sont stockées dans localStorage du navigateur (pas de serveur).
5) Pour partager o publicar, usa GitHub Pages, Netlify ou un hébergeur statique gratuito.

Besoin de ajuda?

  * subir no GitHub Pages,
  * melhorar textos em francês,
  * criar a página de divulgação (Instagram, TikTok),

Observações:
- Este é um protótipo gratuito e offline. Para features avançadas (sincronização, login, analytics, IA conversacional), será preciso back-end e possivelmente custos.
